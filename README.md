# Rule-Based Chatbot in Python

## Overview

This project is a simple rule-based chatbot developed using Python. The chatbot responds to predefined user inputs and provides basic conversational interactions. It is designed to demonstrate the fundamentals of conditional statements, loops, and user input handling in Python.

## Features

* Greets users with friendly responses
* Answers common questions
* Handles multiple predefined commands
* Simple and easy-to-understand code
* Exit option to end the conversation

## Technologies Used

* Python 3

## How It Works

The chatbot takes user input and matches it against predefined responses using conditional statements. Based on the user's message, it returns an appropriate reply.
